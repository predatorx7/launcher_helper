PaletteFilter typedef

bool PaletteFilter (HSLColor color)

Hook to allow clients to be able filter colors from selected in a PaletteGenerator. Returns true if the color is allowed.

See also:

  • PaletteGenerator.fromImage, which takes a list of these for its filters parameter.
  • avoidRedBlackWhitePaletteFilter, the default filter for PaletteGenerator.
